从零开始安装VitePress ​

作者：青空
字数：0 字     时长：0 分钟
更新于：1970/1/1 08:00:00

前期工作 ​

  • Node.js 22 及以上版本。我使用的是24.18.0LTS版本
  • 安装命令行CLI工具并进行换源。我使用的是pnpm淘宝源
  • Markdown 语法编辑器
  • VsCode，Git

安装 ​

VitePress可以单独使用，也可以安装到现有项目中。

在这两种情况下，都可以使用以下方式安装它：

shell
pnpm add -D vitepress
1
shell
npm i -D vitepress
1
shell
yarn add -D vitepress
1
shell
bun add -D vitepress
1
如果你遇到了esbuild报错

命令运行之后按Enter

shell
pnpm approve-builds
1

快捷启动（非必须） ​

当前我们需要运行pnpm docs:dev来启动VitePress，这显然不友好。

通过修改package.json，我们可以简化这个操作

json
{
  "devDependencies": {
    "vitepress": "^1.6.4"
  },
  "scripts": {
    "docs:dev": "vitepress dev docs", 
    "docs:build": "vitepress build docs", 
    "docs:preview": "vitepress preview docs"
    "dev": "vitepress dev docs", 
    "build": "vitepress build docs", 
    "preview": "vitepress preview docs"
  }
}
