# 从零开始安装VitePress

## 前期工作

- [Node.js](https://nodejs.org/zh-cn/download) 22 及以上版本。我使用的是`24.18.0LTS`版本
- 安装命令行`CLI`工具并进行换源。我使用的是`pnpm`淘宝源
- Markdown 语法编辑器
- VsCode，Git

## 安装

`VitePress`可以单独使用，也可以安装到现有项目中。

在这两种情况下，都可以使用以下方式安装它：

::: code-group
```shell [pnpm]
pnpm add -D vitepress
```

```shell [npm]
npm i -D vitepress
```

```shell [yarn]
yarn add -D vitepress
```

```shell [bun]
bun add -D vitepress
```
:::

:::: details 如果你遇到了esbuild报错

命令运行之后按`Enter`

::: code-group
```shell [pnpm]
pnpm approve-builds
```
:::
::::

## 快捷启动（非必须）

当前我们需要运行`pnpm docs:dev`来启动`VitePress`，这显然不友好。

通过修改`package.json`，我们可以简化这个操作

::: code-group
```json [package.json]
{
  "devDependencies": {
    "vitepress": "^1.6.4"
  },
  "scripts": {
    "docs:dev": "vitepress dev docs", // [!code --]
    "docs:build": "vitepress build docs", // [!code --]
    "docs:preview": "vitepress preview docs" // [!code --]
    "dev": "vitepress dev docs", // [!code ++]
    "build": "vitepress build docs", // [!code ++]
    "preview": "vitepress preview docs" // [!code ++]
  }
}
```
:::

完成之后就可以通过`pnpm dev`来快速启动`VitePress`，命令运行之后按O快速打开网页

:::: details 通过bat脚本快速启动（Linux使用sh脚本）
将以下命令保存为脚本即可快速完成其功能
::: code-group
```bat [dev.bat]
pnpm dev
```

```bat [build.bat]
pnpm build
```

```bat [preview.bat]
pnpm preview
```

```bat [install.bat]
pnpm install
```
:::
